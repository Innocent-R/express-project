Project: Chapter Chat is a full-stack web application that allows users to share their favorite books. Other users can like or dislike the shared books, creating an interactive reading community.

How It's Made: 

Tech used: HTML, CSS, JavaScript, Node.js, Express.js and MongoDB.

Lessons Learned: 

While building this project, I learned how to manage user interactions, implement database operations for storing and retrieving data.
